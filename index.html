Entropy

The Entropy project is a small function library for various tools useful for entropy calculations.

It also contains a simple GUI for fast text analysis, using all of the provided tools.

WikiTranslator

The WikiTranslator is a semi-automatic translation tool designed for the Official Team Fortress Wiki. It translates sentences commonly found on certain pages among with other automatic checks. Until now, it is available for seven languages and is updated regulary.

It is written in Python 3 and uses Tkinter as the GUI framework to make sure it can run on every computer with a functioning Python installation.

Operation Cleanup

Operation Cleanup is an ongoing project dedicated to the Official Team Fortress Wiki. Its main goal is to increase the consistency among translated articles.

It is written in Python 3 and tries to fetch the several thousand pages as efficient as possible.
